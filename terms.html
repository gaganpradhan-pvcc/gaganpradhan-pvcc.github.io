Operating Systems Terminology
Term Definition
Network Classification
  • LAN - Local Area Network
  • WAN - Wide Area Network
  • MAN - Metropolitan Area Network
Network Models
  • Bus Network - A network topology in which all devices share a common communication pathway, called a "bus"
  • Star Network - A network topology where all devices are connected to a central hub or switch
Protocols
  • CSMA/CD Ethernet - A networking protocal used in ethernet networks where devices listen to the network before transmitting to avoid collisions, and if a collision occurs, a vackoff algorithm is employed
  • CSMA/CA Wifi - A protocol used in Wi-Fi networks where devices listen to the channel before transmitting, and if the channel is busy, they use a random backoff mechanism to try again
Combining Networks:
  • Repeater - A network device that amplifies and retransmits signals to extend the reach of a network
  • Bridge - A device that connects and filters traffic between two or more network segments
  • Switch - A network device that uses MAC addresses to forward data only to the specific device in the network segment that needs it
  • Router - A network device that forwards data between different networks
  • Gateway - A device that connects different networks
Client/Server Model
  • P2P: Peer to Peer - A network architecture where each device has equal status and can act as both a client and a server
  • Distributed System - A system in which components are distributed across multiple computers that work together as a unified whole
  • Cloud Computing - The delivery of computing services, including storage, processing power, and applications
Interent Adressing
  • Tier 1 - Top-level backbone providers
  • Tier 2 - Regional internet providers
  • Tier 3 - Local internet providers
IP Adress
  • ICANN - The Internet Corporation for Assigned Names and Numbers, responsible for managing domain names and IP addresses
  • DNS - Domain Name System, a system that translates human-readable domain names into IP addresses
  • TLD: Top-Level Domain - The highest level in the hierarchical domain naming system
  • Country-Code TLD - A top-level domain assigned to a specific country or territory
Internet Protocols
  • FTP - File Transfer Protocol
  • HTTP - Hypertext Transfer Protocol
  • SMTP - Simple Mail Transfer Protocol
World Wide Web
  • URL - Uniform Resource Locator
  • Client-Side Activities - Actions and processes that occur on the user's device in a client-server architecture
  • Server-Side Activities - Actions and processes that occur on the server in a client-server architecture
Internet Four-Layer Software
  • Application Layer - The top layer of the OSI model, responsible for providing network services directly to end-users or applications
  • Transport Layer - The layer of the OSI model responsible for end-to-end communication and data flow control
  • Network Layer - The layer of the OSI model responsible for logical addressing, routing, and path determenation
  • Link Layer - The lowest layer of the OSI model, responsible for the physical connection between devices and error detection
Internet Is:
  • Packet-Switchting Network - A network in which data is broken into packets for effcient transmission and is routed independently to its destination
  • Fault Tolerant - The ability of a system to continue operating in the presence of hardware or software faults
  • Redundant - Having backup components or systems to ensure continued operation in-case of failure
Cybersecurity
  • Malware - Malicious software deisgned to harm or exploit computer systems
  • Virus - Malicious software that replicates itself by attaching to other programs
  • Worm - Self-replicating malware that spreads independently of other programs
  • Trojan Horse - Malware disguised as legitimate software to decieve users
  • Spyware - Software that secretly monitors and collects information about a user's activities
  • Phishing - A cyberattack method that tricks individuals into revealing sensitive information by pretending to be a trustworthy entity
  • DDoS Attack - Distributed Denial of Service Attack, an attempt to make a machine or a network resource unavailable to users by overwhelming it with traffic
Protection
  • Firewall - A network security system that monitors and controls incoming and ougoing network traffic based on predetermined security rules
  • Spam Filter - A software that identifies and filters out unwanted and unsolicited email (spam)
  • Proxy Server - A server that acts as a intermediary between a user's device and the internet
  • Anti-Virus Software - Software designed to detect, prevent, and remove malicious software from a computer or network
  • Cryptography - The practice and study of techniques for securing communication and data
  • Public Key Encryption - A crpytographic system that uses pairs of keys (public and private) for secure communication
